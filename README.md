# Library Book Inventory Manager

## Overview
Library Book Inventory Manager is a Python-based command-line application designed to manage a library's book inventory efficiently. It allows users to add, search, issue, return, and delete books while storing data persistently using a JSON file.

## Features
- Add new books
- Search books by title or author
- Issue books
- Return books
- Display all books
- Generate inventory reports
- Delete books
- Automatic JSON data storage
- Automatic data loading on startup

## Technologies Used
- Python
- JSON
- Object-Oriented Programming (OOP)
- File Handling
